VacinAPI

 

 

O que é:

VacinAPI é uma API REST de controle de vacinação. Feita utilizando Java, Spring Boot, Hibernate e MySQL.

 

Resources:

VacinAPI tem apenas 2 endpoints.

/cadastros/usuario - para acessar a lista de usuários

/cadastros/vacinas - para acessar a lista de aplicação de vacina

 

Rotas: 

GET /cadastros/usuario

POST /cadastros/usuario - Parâmetros requeridos: nome do usuário, email válido cadastrado, cpf válido, data de nascimento (qualquer data até o dia atual).

GET /cadastros/vacinas

POST /cadastros/vacinas - Parâmetros requeridos: nome da vacina, email de um usuário ja cadastrado, data de vacinação (qualquer data até o dia atual).
